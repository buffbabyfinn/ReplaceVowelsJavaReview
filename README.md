# Vowel Replace

#### This site takes user input and replaces vowels with dashes , Feb 12, 2016.

#### By _**Megan Fayer**_

## Description

_The user inputs a word, phrase, or sentence, and the pages replaces all instances of vowels, lower and upper case, with dashes.._
